# 任务：构建康乃尔公司介绍单页落地页

## 你需要做的事

读取以下两个文件，然后构建一个单页滚动落地页：

1. **DESIGN.md** — 视觉规范（色板、字体、间距、动效、禁止事项）
2. **CONTENT.md** — 内容文案（24 个区块的完整文案）

## 关键要求

- 输出单个 `index.html` 文件，所有 CSS 和 JS 内嵌
- 这是用网页形式替代 PPT 讲解的企业介绍落地页，不是传统网站
- 浅色洁净制造风格：白底 + 深蓝标题 + 大量留白
- 24 个滚动区块，按 CONTENT.md 中的顺序排列
- 顶部固定导航栏，滚动时高亮当前区块
- 滚动入场动画（淡入上移）
- 大数字递增动画
- 卡片 hover 效果
- 白/浅灰底色交替
- 完全响应式

## 素材

`assets/` 目录下有 25 个图片文件，按子目录分类：
- `assets/company/` — 厂区、产线、实验室、精修片
- `assets/tpu/` — TPU 产品图（微结构、传感器贴、敷料、拉伸、绷带、定制流程）
- `assets/maxiscrub/` — MaxiScrub logo + 结构图
- `assets/aquachill/` — AquaChill 结构图、材料原照、场景图
- `assets/quality/` — 质量控制流程图
- `assets/supply/` — 供应链流程图
- `assets/contact/` — 合作流程图

在对应的区块中使用这些真实素材。如果某个区块需要配图但 assets/ 中没有合适的，用 $imagegen 生成。

## 保密红线（不可违反）

- 不写客户名称（丸红、斯迪克、宝洁、恒鹏、爹地宝贝、海氏海诺等）
- 不宣称医疗功效（不写"促进愈合""抗菌消炎""治疗"）
- 不用占位图、不编造数据
- 不用 emoji、不用渐变文字、不用 hero eyebrow/kicker/badge/pill

## 验收标准

- 浏览器打开 index.html，从头滚到尾是一个完整的公司介绍
- 所有 24 个区块内容完整呈现
- 数据准确，与 CONTENT.md 一致
- 移动端可正常浏览
- 视觉效果达到专业级 B2B 企业介绍水准
